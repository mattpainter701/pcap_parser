About

This tool visualizes network activity captured in PCAP files and processed into CSV format.

The visualization shows relationships between MAC addresses and IP addresses with port information.

Features:

  • Interactive network graph visualization
  • Drag-and-drop node positioning
  • Zoom and pan controls
  • Node filtering and search
  • Hover for detailed information
